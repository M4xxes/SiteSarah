## Animarah – Massage équin & communication animale

### Structure du projet

- `index.html` : page d’accueil (présentation, valeurs, formulaire de contact).
- `massages.html` : présentation détaillée du massage équin.
- `communication.html` : explications sur la communication animale.
- `formules.html` : ensemble des formules et tarifs (massages, suivis, communication, concours).
- `assets/css/styles.css` : feuille de styles principale (mise en page, typographie, couleurs).
- `assets/img/logo-animarah.png` : logo Animarah utilisé dans le header.

### Utilisation

#### Avec Vite (recommandé)

1. Dans le dossier `sarahSite`, installer les dépendances si ce n’est pas déjà fait :
   - `npm install`
2. Lancer le serveur de développement :
   - `npm run dev`
3. Ouvrir le navigateur sur l’URL indiquée par Vite (par défaut `http://localhost:5173`), la page d’accueil `index.html` s’ouvre automatiquement.

#### Sans tooling

Ouvrir directement `index.html` dans un navigateur, ou lancer un petit serveur statique (par exemple avec l’extension Live Server ou `python -m http.server`) à la racine du dossier `sarahSite`.
